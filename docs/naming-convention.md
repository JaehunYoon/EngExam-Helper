# Naming Rule

## data/ words as .json

Follow this rule to add English tests to 'data' directory in json format.

### Format

```tree
▼ data/
  ▼ dsm{years}/
    ▼ {semester}/
      {sequence_no}.json
      ...
```

### Example

```tree
▼ data/
  ▼ dsm2017/
    ▼ 2nd/
      01.json
      02.json
      03.json
      04.json
```
